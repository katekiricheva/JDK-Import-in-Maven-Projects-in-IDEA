# Bug Report: Incorrect Project SDK Value

## Summary
TC01-TC12 Project SDK is set incorrectly after the Maven project import in IntelliJ IDEA.

## Detailed description
The Project SDK value is intended to reflect the Java version defined in the Maven `pom.xml` file. 

## Pre-conditional
The Maven project is freshly imported and opened in IntelliJ IDEA.

## Steps to reproduce
1. Open the `pom.xml` file (parent `pom.xml` for multi-module projects).
2. Verify that the Java version is correctly defined there.
3. Navigate to: File → Project Structure → Project.
4. Check the "SDK" value.

## Actual result
The Project SDK value is set to the newest JDK available on the system.

## Expected Result
The Project SDK value is set to the Java version specified in the `pom.xml` file (parent `pom.xml` for multi-module projects).

## Priority
High

## Severity
Important
