# Bug Report: Incorrect Module SDK Value

## Summary
TC01-TC12 Module SDK is set incorrectly after the Maven project import in IntelliJ IDEA.

## Detailed description
The Module SDK value is intended to reflect the Java version defined in the module’s `pom.xml` file. 

## Pre-conditional
The Maven project is freshly imported and opened in IntelliJ IDEA.

## Steps to reproduce
1. Open the `pom.xml` file for all modules.
2. Verify that the Java version is correctly defined there.
3. Navigate to: File → Project Structure → Modules → Dependencies.
4. Verify the "Module SDK" settings.

## Actual result
The Module SDK value for all modules is set to "Project SDK" with the newest Java version available on the system.

## Expected Result
The Module SDK value is set to the Java version specified in the module’s own `pom.xml` file.

## Priority
High

## Severity
Important
